SIGNIFICADO DAS TAGS BASICAS DO HTML

A Tag "HTML" Define o início e o fim do documento HTML

A Tag "HEAD"

Contém metadados, como título, links para estilos, scripts e codificação de caracteres
Define o título da página

A Tag "Title" define o título da página (aparece na aba do navegador)

A Tag "BODY" Contém o conteúdo visível da página

A Tag "H1"até"H6"significa TÍTULOS, sendo "H1" o mais importante e "H6"o menos

A Tag "Div" Significa divisão genérica de bloco

A tag div é uma divisão virtual dentro das paginas html

A Tag "HR!

A tag hr é uma linha criada dentro da página

A Tag "IMG"


A tag img permite colocar imagem na página

A Tag "INPUT"

A tag input serve para o usario digitar alguma coisa

A Tag "BUTTON"

A tag button significa um botão para o usuário clicar

A Tag "BR"


Quebra a linha

A Tag "P"

Siginifica parágrafo de texto

A Tag "STRONG"

Deixa o texto em negrito (semântico)

A Tag "B"

Deixa o texto em negrito (não semântico)

A Tag "EM"

Deixa o texto em itálico (semântico)

A Tag "I"

Deixa o texto em itálico (não semântico)

A Tag "UL"

Lista não ordenada (com marcadores)

A Tag "OL"

Lista ordenada (numerada)

A Tag "LI"

Item da lista

A Tag "VIDEO"

Insere um vídeo

A Tag "AUDIO"

Insere um áudio

A Tag "TABLE"

Define uma tabela

A Tag "TR"

Define uma linha da tabela

A Tag "TD"

Define uma célula (dado)

A Tag "TH"

Define uma célula de cabeçalho

A Tag "FORM

Define um formulário

A Tag "LABEL"

Rótulo para um campo de entrada

A Tag "TEXTAREAL"

Caixa de texto para entrada maior

A Tag "SELECT"

Lista suspensa (dropdown)

A Tag "OPTION"

Opção dentro de (select)
